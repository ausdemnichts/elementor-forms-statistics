Elementor Forms Statistics is a plugin for analyzing form submissions in Elementor.

== Elementor Compatibility ==

* Tested with Elementor 3.34.0 and Elementor Pro 3.34.0.

== Description ==

Elementor Forms Statistics is a plugin that allows editors to view all submissions received through Elementor forms. It offers detailed statistics, displayed as charts and tables, providing valuable insights into form submission trends.

**Main Features:**

* Real-time access to form submissions.
* Visual statistics and chart generation for submission data.
* Tabular views of all Elementor form submissions.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/mdp-elementor-forms-extras` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Enjoy
